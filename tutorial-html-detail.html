Belajar HTML #02: Apa itu HTML?

Ditulis oleh Pusat Solusi | Diperbarui pada 24 September 2025

Selamat datang di seri tutorial HTML dari Pusat Solusi. Pada bagian kedua ini, kita akan berkenalan lebih dalam dengan HTML. Apa sih sebenarnya HTML itu?

HTML adalah singkatan dari Hypertext Markup Language. Sesuai namanya, HTML adalah bahasa markup, bukan bahasa pemrograman. Fungsinya adalah untuk menstrukturkan konten di halaman web.

Bayangkan halaman web sebagai sebuah dokumen. HTML-lah yang memberi tahu browser, "ini adalah judul", "ini adalah paragraf", "ini adalah gambar", dan sebagainya. Tanpa HTML, halaman web hanya akan menjadi teks biasa yang tidak terstruktur.

Analogi Kerangka Manusia

Contoh Kode HTML

Berikut ini adalah contoh kode HTML yang sangat sederhana:

example.html 
<!DOCTYPE html>
<html lang="id">
<head>
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Ini adalah Judul Utama</h1>
    <p>Ini adalah sebuah paragraf.</p>
</body>
</html>

Jangan khawatir jika Anda belum memahaminya. Kita akan membahas setiap bagian dari kode di atas pada tutorial-tutorial berikutnya. Intinya adalah, kode tersebut memberitahu browser cara menampilkan konten di dalamnya.
